# GeiserX Runtipi App Store

Community app store for [Runtipi](https://runtipi.io/) with self-hosted applications by [GeiserX](https://github.com/GeiserX).

## How to Use

1. Open your Runtipi dashboard
2. Go to **Settings > App Stores > Add App Store**
3. Paste this repository URL: `https://github.com/GeiserX/runtipi-appstore`
4. Give it a name (e.g., "GeiserX Apps") and save
